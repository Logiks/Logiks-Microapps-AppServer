# Recommended Prerequisites

# 1. Basic Technical Understanding (Mandatory)

### Computer & Web Basics

* Understanding of:

  * Browser usage
  * URLs
  * Forms
  * Tables
  * File upload/download
  * Session/login concepts

### Expected Outcome

Should comfortably navigate and test web applications.

---

# 2. HTML/CSS Fundamentals (Mandatory)

### Topics

* HTML tags
* Forms
* Tables
* Div/layout basics
* CSS classes
* Basic responsive concepts

### Why Needed

Because:

* Reports
* Form layouts
* Custom components
* Dashboard UI
  all require UI understanding.

### Minimum Skill

Should be able to:

* Create a simple form
* Create a table
* Style a button/input

---

# 3. JavaScript Fundamentals (Very Important)

### Topics

* Variables
* Functions
* Conditions
* Loops
* Events
* DOM basics
* Async/AJAX basics
* JSON

### Why Needed

Training includes:

* Script execution
* Dynamic forms
* API calls
* Hooks
* Automator
* Custom formatters

### Minimum Skill

Should understand:

```js
fetch()
if/else
functions
event handling
```

---

# 4. API Understanding (Important)

### Topics

* What is an API
* GET/POST requests
* JSON request/response
* Headers
* Authentication basics

### Why Needed

Most modules rely on:

* API integrations
* Dropdown API calls
* Notification APIs
* Webhooks
* Post-submit hooks

### Minimum Skill

Should be able to:

* Test API using Postman
* Read JSON response

---

# 5. Database Fundamentals (Important)

### Topics

* Tables
* Rows/columns
* Primary key
* Relationships
* CRUD operations

### Optional but Helpful

Basic SQL:

```sql
SELECT
INSERT
UPDATE
DELETE
```

### Why Needed

Required for:

* Reports
* Excel imports
* Aggregate functions
* Data mapping

---

# 6. AJAX & Dynamic UI Concepts (Recommended)

### Topics

* Dynamic dropdowns
* Partial refresh
* Async loading
* Chained fields

### Why Needed

Used heavily in:

* Contractor → Site dependency
* Searchable dropdowns
* Dynamic reports

---

# 7. Basic Debugging Skills (Recommended)

### Topics

* Browser console
* Network tab
* Reading errors
* Logs

### Why Needed

Essential for:

* API debugging
* Form debugging
* Script troubleshooting

---

# 8. Framework Familiarity (Optional but Ideal)

### Should Know

* Existing framework navigation
* Module structure
* Existing project flow

### Helpful Areas

* Existing forms
* Existing reports
* Existing hooks

---

# Ideal Candidate Profile

An ideal trainee should:

* Be comfortable with coding basics
* Understand frontend interaction flow
* Have basic backend/API awareness
* Be willing to debug independently
* Be able to read existing code and extend it

---

# Biggest Success Factors

The most important skills for succeeding in this training are:

1. JavaScript understanding
2. API integration knowledge
3. Debugging capability
4. Logical thinking for workflows
5. Ability to understand event-driven systems

These will directly impact success in:

* Hooks
* Automation
* Dynamic forms
* Notifications
* Reports
* Custom scripting
